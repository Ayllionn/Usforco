Bonjour ! 
tout se beau monde est la pour aidé a developpé des bots discord en python. 


Pour ce programme, il est conçu pour interagir uniquement avec les serveurs Discord de base.
Toutefois, cela peut varier en fonction des extensions que vous installez et de la source utilisée. Il est donc important de garder cela à l'esprit.


Avant de commencer,
assurez-vous d'avoir installé Python et que pip est fonctionnel dans votre terminal.
De plus, notez que certaines bibliothèques peuvent nécessiter les permissions administrateur pour être installées si elles ne le sont pas déjà.

main.py -> Pour les 1er lancement du bot passer par ce fichier. Il va config et vous permetre d'accédé a des tools dans un terminal comme la cmd `config`
starter.py -> Permet de lancer le bot. Vous pouvez passez par le main.py avant car le bot a besoin d'un config.json pour trouvé un token
